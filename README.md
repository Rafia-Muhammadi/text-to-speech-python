# 🎙️ Text to Speech Converter (Python GUI)

A simple and interactive Text-to-Speech (TTS) application built using Python and Tkinter.

## 🚀 Features
- Convert text into speech
- Adjust voice (Male/Female)
- Control speed and volume
- Save speech as MP3 file
- Play audio using pygame
- Simple and user-friendly GUI

## 🛠️ Tech Stack
- Python
- Tkinter
- pyttsx3
- gTTS
- pygame

## ▶️ How to Run

1. Clone the repository:
  https://github.com/Rafia-Muhammadi/text-to-speech-python.git
2. Install dependencies:
 pip install pyttsx3 gtts pygame
3. Run the program:

python main.py
